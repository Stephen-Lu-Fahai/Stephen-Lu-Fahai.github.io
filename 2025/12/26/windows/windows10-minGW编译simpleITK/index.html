二、依赖

1、Lua

要求版本: 5.3 到 5.5之间，选择：lua-5.4.8.tar.gz
编译

当前目录会生成一个install的安装目录，把src下编译出来的dll库文件拷贝到install/bin 目录下，然后把bin目录加到系统环境变量。
验证：能在命令行正确执行lua.exe 命令就行。

2、swig

minGW无法编译，直接下载可执行文件，选择4.4.1版本，解压并将目录添加到环境PATH。

3、itk

选择版本 InsightToolkit-5.4.5.zip
直接使用cmake-gui配置相关路径，正常编译安装即可。

问题1： ITK build directory path length is too long

直接修改一下build的路径长度，缩短文件名称，重新编译即可。

google了一下，应该和minGW的版本有关，但这只是一个返回的错误状态，不影响真正的代码。
