二、配置

在cmake-gui中选择vtk源码路径和编译路径，随后选择minGW Makefile并指定响应的gcc路径。

在配置界面中选择对应的几个选项：

1、安装路径

根据需求在CMAKE_INSTALL_PREFIX的一个安装目录即可。

2、编译选项

在选项CMAKE_CXX_FLAGS 中填入 -Ofast -flto
否则在make过程中会出现报错： Cmake qt ,vtkDataArray.cxx.obj: File too big

3、QT路径

如果项目中需要QT支持，可以勾选

并且需要在选项Qt5_DIR 中选择QT安装目录下的mingw的cmake库目录。
