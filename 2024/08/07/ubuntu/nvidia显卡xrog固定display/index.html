nvidia显卡xrog固定display

2024-08-07 ubuntu系统 ubuntu系统 0 Comments Word Count: 700(words) Read Count: 3(minutes)

现象描述

在之前的文章 “nvidia显卡xorg多种配置方案” 中可以把显卡的多个视频输出口配置成独立的xorg。默认情况下，无论实际接的是哪个输出口，都会把第一个检测到的显示器设置为 display :1.0， 随后是 :1.1 …
这样做的好处是多个显示器会自动排序。 但也有一个缺点：如果开机的时候只接入一个显示器，系统运行后再接入另一个显示器，那么新接入的就不是独立的xorg，而是被并入第一个display中，成为扩展。

为了支持热插拔，可以增加 UseDisplayDevice 选项，配合allowdEmpty使用，这Disply就不是从1.0开始，而是每个输出口对应固定的display。 但是好处是新接入的显示器也会运行再独立xorg上，可以完美支持热插拔。
